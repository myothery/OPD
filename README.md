# 🔵 OPD — Circles Generator & Processor (Go)

## 📌 Описание

Проект на языке **Go**, который:

- генерирует случайные окружности без пересечений  
- сохраняет их параметры в JSON  
- визуализирует окружности на графике  
- вычисляет диаметры по трём точкам  
- группирует результаты и выводит статистику  

---

## 🚀 Возможности

- 📍 Генерация окружностей с ограничениями  
- 📄 Экспорт в JSON  
- 📊 Построение графика (`gonum/plot`)  
- 📐 Вычисление диаметра по 3 точкам  
- 📈 Группировка и сортировка данных
