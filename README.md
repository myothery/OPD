# 🔵 OPD — Circles Generator & Processor (Go)

## 📌 Описание проекта

Проект на языке **Go**, который состоит из двух основных частей:

1. **Генератор данных** — создает случайные окружности без пересечений, визуализирует их и сохраняет параметры в JSON-файл.  
2. **Обработчик данных** — считывает JSON, вычисляет диаметры окружностей по трём точкам и выводит итоговую статистику.

Проект демонстрирует использование:

- структур данных  
- JSON-файлов  
- математических вычислений  
- геометрических алгоритмов  
- сортировки  
- визуализации данных  
- обработки ошибок  

---

## 🚀 Возможности

- 📍 Генерация окружностей в заданном диапазоне

## 📦 Структуры данных

### Point

```go
type Point struct {
    X float64
    Y float64
}
```

Точка на двумерной координатной плоскости.

Используется для хранения координат:

X — положение по оси X
Y — положение по оси Y

Применяется для задания точек на окружности и выполнения геометрических вычислений.

### Circle
```go
type Circle struct {
    X, R float64
}
```

Структура окружности.

Содержит:

X — координату центра окружности по оси X
R — радиус окружности

Используется на этапе генерации окружностей.
### CircleData
```go
type CircleData struct {
    CenterX float64
    Radius  float64
    Points  []Point
}
```

Структура для сохранения данных в JSON-файл.

Содержит:

координату центра
радиус
массив из трёх точек окружности

## ⚙️ Генерация окружностей

Файл: `generator/main.go`

### Что делает программа

После запуска генератор:

- создает заданное количество случайных окружностей  
- размещает их без пересечения  
- выбирает 3 случайные точки на верхней части каждой окружности  
- сохраняет результат в `circles.json`  
- строит изображение `circles_with_points.png`

---

### 🔥 Основной алгоритм — `generateCircles()`

Для каждой новой окружности случайно выбираются:

- радиус  
- положение центра по оси X  

После этого выполняется проверка, не пересекается ли новая окружность с уже созданными.

### Проверка пересечения

```go
dx := math.Abs(x - c.X)
minDist := 2 * math.Sqrt(r*c.R)
```

Где:

dx — расстояние между центрами
minDist — минимально допустимое расстояние между окружностями

Если:

dx < minDist

то новая окружность отклоняется.

### 🎯 Построение окружности

Используется параметрическая формула:

x = x0 + r*cos(t)
y = r + r*sin(t)

По множеству точек строится линия круга.

### 🔴 Генерация точек на окружности

Функция:

```go
generateTopPoints()
```
Создает 3 случайные точки на верхней части окружности.

Используется диапазон углов:

0 ≤ t ≤ π

Это соответствует верхней половине круга.

💾 Формат JSON
```json
[
  {
    "CENTER_X": 15.2,
    "RADIUS": 7.4,
    "TOP_POINTS": [
      { "X": 10.1, "Y": 12.3 }
    ]
  }
]
```

📈 Визуализация

Используется библиотека:
```go
gonum.org/v1/plot
```
На изображении отображаются:

окружности разными цветами
точки на окружностях

Результат сохраняется в файл:

`circles_with_points.png`

## ⚙️ Обработка данных

Файл: `process/main.go`

### Что делает программа

После запуска программа:

1. Запрашивает имя JSON-файла  
2. Загружает данные  
3. Для каждой окружности берет 3 точки  
4. Вычисляет диаметр  
5. Группирует одинаковые диаметры  
6. Выводит итоговую статистику  

---

### 🔥 Основной математический блок — `diameterFromPoints()`

По трём точкам окружности вычисляется:

- центр окружности  
- радиус  
- диаметр  

Сначала определяется центр:

```go
(xc, yc)
```
Затем вычисляется радиус как расстояние от центра до точки.

После этого:
```go
diameter = radius * 2
```

⚠️ Проверка ошибки

Если три точки лежат на одной прямой:
```go
determinantA == 0
```
то окружность построить невозможно.

### 🔄 Округление результатов

Используется функция:

```go
roundDiameter()
```
Она нужна для корректной группировки чисел с плавающей точкой.

Пример:

10.000001 → 10.00
9.999999  → 10.00
📊 Группировка данных

Используется словарь:
```go
map[float64]int
```
Пример:

10.00 → 3
14.20 → 2
20.00 → 1

Это означает количество окружностей каждого диаметра.

📌 Сортировка

Для вывода по возрастанию используется:
```go
sort.Float64s()
```
